Висновок 

У ході цієї лабораторної роботи було розглянуто та реалізовано алгоритм Флойда для вирішення задачі пошуку найкоротших шляхів у графах. Ключовою метою було освоєння методів паралельної обробки графів.

Для цього:

Було розроблено та протестовано послідовну версію алгоритму Флойда.

Було спроектовано та реалізовано паралельну версію алгоритму з використанням технології MPI (Message Passing Interface).


Особливу увагу було приділено методам розподілу даних (матриці суміжності) між різними процесами для ефективного розпаралелювання обчислень.

Робота завершилася проведенням обчислювальних експериментів для порівняння часу виконання послідовного та паралельного алгоритмів, що продемонструвало переваги паралельного підходу.
